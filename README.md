# University Project Portfolio

Добро пожаловать в мой академический портфолио-репозиторий. Здесь собраны лучшие курсовые и практические работы, выполненные мной в процессе обучения в Сибирском федеральном университете (СФУ) по направлению «Бизнес-информатика».

Этот репозиторий демонстрирует мои навыки в **веб-разработке**, **моделировании бизнес-процессов**, **проектировании баз данных (SQL)** и **аналитике данных (BI)**.

---

## 📂 Содержание репозитория

### 1. Веб-разработка (Web-разработка)
*   **Файл:** `Web-Development-Coursework`
*   **Стек:** PHP, MySQL, HTML5, CSS3, JavaScript, OpenServer.
*   **Описание:** Разработка полноценного веб-приложения-портфолио с системой авторизации/регистрации пользователей, разграничением прав доступа (роли администратора и обычного пользователя), возможностью управления проектами (CRUD) и модулем комментариев. Реализована адаптивная верстка.

### 2. Моделирование бизнес-процессов
*   **Файл:** `Business-Process-Modeling`
*   **Стек:** Нотации IDEF0, DFD, IDEF3, BPMN 2.0, ARIS.
*   **Описание:** Комплексное моделирование бизнес-процесса потребительского кредитования на примере ПАО «Сбербанк». Разработана функциональная модель, диаграммы потоков данных, сценарии выполнения и организационная структура. Демонстрирует навыки процессного анализа и составления технических требований.

### 3. Проектирование базы данных (PostgreSQL)
*   **Файл:** `PostgreSQL-Database-Design`
*   **Стек:** PostgreSQL, DBeaver, SQL, ER-диаграммы (Чена/Мартина).
*   **Описание:** Разработка макета реляционной базы данных для автоматизации работы сервисного центра по ремонту компьютерной техники. Включает в себя создание логической и физической схем, определение связей между таблицами, нормализацию данных, написание сложных SQL-запросов (JOIN, агрегации) и создание представлений (Views).

### 4. Визуализация данных и бизнес-аналитика (Power BI)
*   **Файл:** `Data-Visualization-PowerBI`
*   **Стек:** Power BI, Loginom (первичный анализ), DAX, датасет Brazilian E-Commerce (Kaggle).
*   **Описание:** Проведение полного цикла ETL-анализа данных: оценка качества данных, преобразование типов, создание вычисляемых метрик и построение набора интерактивных дашбордов. Решены две бизнес-задачи: анализ географии продаж для разработки стратегии расширения, а также аудит лояльности клиентов и влияния логистики на повторные покупки.

---

## 🛠️ Ключевые навыки в рамках проектов

*   **Базы данных:** PostgreSQL, MySQL, написание SQL-запросов.
*   **Визуализация:** Power BI, построение дашбордов и отчетов.
*   **Анализ данных:** Статистический анализ (Loginom), работа с реальными коммерческими датасетами.
*   **Веб-технологии:** Fullstack-разработка (PHP + HTML/CSS + MySQL).
*   **Бизнес-анализ:** Моделирование в нотациях BPMN, IDEF0, DFD, ARIS.
